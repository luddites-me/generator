## `package.json` scripts

`yarn build`

- Sorts JSON and runs documentation

`yarn docs:sync-readme`

- Runs the documentation tools

`yarn pretest`

- Runs ESLint

`yarn sortJson`

- Alpha-sorts all JSON files

`yarn test`

- Runs the Jest tests
